Tags em HTML

As tags são os elementos fundamentais do HTML, usadas para estruturar o conteúdo de uma página web. Elas são compostas por uma tag de abertura e, em muitos casos, uma tag de fechamento.

Por exemplo, a tag <p> é usada para definir um parágrafo, enquanto a tag <h1> define um cabeçalho de nível 1.
